# First-CEO-iOS
첫사장 : 초보 사장님을 위한 사업장 근로 환경 진단 도우미

</div>

### 💡Features 
- 사업장 정보에 따른 맞춤형 고용 환경 진단 기능 (업종, 사업장 및 매출액 규모)
- 초보 사업자에게 익숙하지 않을 수 있는 `4대보험`, `임금`, `해고`, `근로계약` 등에 관련된 법령 정보 제공
- 고용 환경 진단 결과에 따라 보완해야 할 요소 리포트
